## Amiga C with Linux

vcm must be executable

```console
chmod +x ~/amiga/C/Linux/vbcc/bin/vca
```

add PATH directory

```console
nano ~/.bashrc
```

add ligne
```console
export PATH=$HOME/amiga/C/Linux/vbcc/bin:$PATH
```

Save ctrl+o , ctrl+x

reload 
```console
source ~/.bashrc
```

go into any c source directory

```console
cd ~/amiga/C/Sources/disk01

vca nomfic.c
```
